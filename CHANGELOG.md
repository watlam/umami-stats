# 更新日志

所有重要的更改都将记录在此文件中。

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [1.0.0] - 2025-01-15

### 新增
- 🎉 首次发布
- ✨ 完整的后台设置页面
  - Umami 实例地址配置
  - 网站 ID 配置
  - API Token 配置
  - 时区选择
  - 缓存时间设置
  - 连接测试功能
- 📊 统计数据仪表盘
  - 今日访客显示
  - 今日访问量显示
  - 本月浏览量显示
  - 总浏览量显示
  - 一键刷新数据
- 📋 嵌入代码生成器
  - 可编辑代码区域
  - 实时预览效果
  - 一键复制代码
  - 样式自定义支持
- 🔌 REST API 端点
  - `/wp-json/umami/v1/stats`
  - 公开访问，无需认证
- 🌐 完整中文界面
  - 简体中文支持
  - 中文提示说明

### 技术细节
- 支持 Umami v3 API
- 数据缓存机制，减少 API 调用
- 响应式后台界面
- 三等分布局设计
